Функция Save_Click вызывается по щелчку на кнопку Save в окнце документа
Сораняет данные, выбранные пользователем в документ Microsoft Word
1.Вызывает фнукцию Data() для сохранения всех введенных пользователем данных в массив data[].
2.Создает экземпляр Word.Document doc
3.Создает объект приложения Word.Application app = new Word.Application()
4.Открывает документ из введенной директории
5.Создает объект wBookmarks, который содержит все закладки
6.В цикле присваивает каждой закладке текст из массива данных пользваотеля data[]
7.Сохраняем в документ новый, именуем его Ф.И.О. пациента
8.Закрываем документ
